# My Web

A futuristic all-in-one personal productivity and finance dashboard built with React, Vite, Framer Motion, Recharts, and lucide-react.

## Features

- Dashboard home with quick stats, savings, tasks, upcoming events, and finance insight cards.
- Smart calendar with daily, weekly, and monthly view modes, recurring labels, completion states, notes, and category chips.
- Expense and finance tracker with manual logging, categories, searchable timeline, bar charts, pie charts, and budget-style summaries.
- Tasks, study, exam, and productivity management with priorities, completion states, categories, and progress analytics.
- Savings and goals tracker with deadlines, adjustable progress, milestones-ready cards, and motivational visual progress bars.
- Customizable widgets with drag-and-drop ordering, section renaming/removal, layout density, and accent color personalization.
- Responsive dark UI with glassmorphism, soft neon glow, smooth animations, and premium dashboard styling.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This repository includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.
After the project is pushed to GitHub, enable Pages with **Source: GitHub Actions** in the repository settings. Every push to `main` will build and publish the dashboard from the repository.
