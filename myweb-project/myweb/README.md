# MyWeb OS 🚀
### Personal Productivity & Finance Management Dashboard

A premium, futuristic all-in-one productivity and finance app built with React.

## Features
- **Dashboard** — Stats, charts, task overview, goals summary
- **Smart Calendar** — Monthly/weekly/daily views, color-coded events
- **Finance Tracker** — Expense logging, pie/bar charts, budget tracking
- **Tasks** — Priority system, categories, deadlines, completion %
- **Goals** — Visual savings goals with progress bars & deposit tracking
- **Analytics** — Cash flow, radar charts, activity insights

## Quick Start
```bash
npm install && npm start
```

## Push to GitHub
```bash
chmod +x push-to-github.sh && ./push-to-github.sh
```

## Tech Stack
React 18 · Recharts · Lucide React · date-fns · localStorage

Built with a futuristic dark glassmorphism UI design system.
